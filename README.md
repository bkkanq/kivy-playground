Install kivy
================

- For mac
  - `python -m pip install --upgrade pip setuptools virtualenv`
  - `source kivy_venv/bin/activate`
  - `python -m pip install "kivy[base]" kivy_examples`
  - ref) https://kivy.org/doc/stable/gettingstarted/installation.html
